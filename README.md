A modern esports-themed website where players can explore tournaments, discover competitive teams, and join gaming communities.

## About The Project

This is a website I made for my ITWM101 course. It shows a professional esports team with pages for the team members, tournaments, and contact info. The site works on phones, tablets, and computers.

## What's Included

- **Home Page** - Shows what Valorzone is about
- **Team Page** - Lists all the team members and their roles
- **Tournaments Page** - Shows upcoming and past tournaments
- **Contact Page** - Has a form to contact the team

## What I Built It With

- HTML5 - For the page structure
- CSS3 - For styling and animations
- Bootstrap - For making it responsive
- JavaScript - For the contact form and animations

## Features

- Works on phones, tablets, and computers
- Contact form that checks if email is correct
- Smooth animations when you scroll
- Team member cards with hover effects
- Tournament list with dates and prizes
- Mobile menu for smaller screens
